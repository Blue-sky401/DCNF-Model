# DCNF Geological Modeling – Minimal Reproducible Example

This repository provides a simplified demonstration of the Deep Clustering Neural Field (DCNF) method for 3D coal-bearing strata modeling using borehole data.

## Requirements
- Python 3.8+
- numpy
- pandas
- scikit-learn

## Synthetic Data Generation
Run `python data/generate_synthetic_data.py` to create a synthetic dataset (because real data are confidential).

## Training a Simplified DCNF Model
Run `python src/train_simple_model.py` to train a lightweight MLP model with soft partition probabilities. Expected RMSE is about 5-8 m.

## GUI Interactive Visualization Tool (Optional)

This repository includes an interactive 3D visualization system for borehole data and geological models. The GUI allows you to:

- Load borehole data from CSV files (e.g., the synthetic data generated by `data/generate_synthetic_data.py`)
- Visualize borehole points, triangulated meshes, and stratigraphic surfaces
- Adjust visual effects (color mapping, transparency, smoothness)
- Perform model slicing and fault handling

### Requirements

The GUI depends on `PyQt5` and `pyvista` (already included in `requirements.txt`).

### How to Run

```bash
python gui/main_gui.py
