# User Guide for DCNF Demonstration

This guide describes how to use the simplified DCNF (Deep Clustering Neural Field) demo.

## Input Data Format

The demo expects a CSV file with the following columns (same as generated by `data/generate_synthetic_data.py`):

- `x` : X coordinate (m)
- `y` : Y coordinate (m)
- `z_true` : True elevation (m) – the target variable
- `layer` : Stratum layer index (0–9)

## Output

- The training script prints the RMSE (Root Mean Square Error) on the test set.

## Customization

- You can replace the synthetic CSV with your own borehole data, as long as the column names match.
- Modify the `soft_prob` function in `train_simple_model.py` to adjust the soft partition probabilities according to your geological assemblages.

## Expected Behavior

On the provided synthetic data, the script runs without errors and outputs an RMSE value typically between 5 and 8 meters. The exact value varies due to random noise.
